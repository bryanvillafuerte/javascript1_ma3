JavaScript MA03
----------------
Level 2

1. In your own words, describe what JavaScript Code Style is.
    JavaScript Code Style is basically how programmers write their codes. This typically covers naming and declaring rules for variables and functions and how programmers use white space, indentation, and comments.

2. What is a Regular Expression?
    Regular expression is a pattern used to match character combinations in strings. In JavaScript, regular expressions are also objects. These patterns are used with the exec and test methods of RegExp, and with the match, replace, search, and split methods of String. This chapter describes JavaScript regular expressions.

3. What does the .random() method do?
    The random() method returns a random number from 0 (inclusive) up to but not including 1 (exclusive).

4. What does the .sort() method do?
    The sort() method is used to sort an array. The sort order can be either alphabetic or numeric, and either ascending or descending.

5. What does the .reverse() method do?
    Given the name, the reverse() method reverses the order of the elements in an array.

6. List two examples of bad JavaScript code.
    Identifiers beginning with a capital letter:
    var Color = "red";

    Not using the var statement to define variables:
    color = "red";

7. What data types can a JSON object contain?
    A JSON object can contain a string, a number, an object (JSON object), an array, a boolean, or null.

8. What is minifying JavaScript Code?
    Minifying refers to the process of removing unnecessary or redundant data without affecting how the resource is processed by the browser - e.g. code comments and formatting, removing unused code, using shorter variable and function names, and so on.

9. What is a global variable?
    A variable declared outside a function, becomes GLOBAL. A global variable has global scope which means all scripts and functions on a web page can access it.
